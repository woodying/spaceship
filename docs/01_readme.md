### **게임 기획서 초안: 냥냥 슈터 (Nyang Nyang Shooter)**

**1. 게임 개요**

*   **장르:** 캐주얼 우주 슈팅 게임
*   **플랫폼:** 모바일, PC
*   **컨셉:** 귀여운 고양이 비행기를 조종하여 우주를 침략하는 쥐 군단을 물리치고 평화를 되찾는 스테이지 클리어 방식의 슈팅 게임.

**2. 게임 특징**

*   **독특한 캐릭터:** 고양이를 모티브로 한 개성 넘치는 비행기와 쥐 형태의 적 캐릭터들.
*   **다양한 스테이지:** 우주의 여러 행성과 공간을 배경으로 한 다채로운 스테이지 구성.
*   **전략적 성장:** 미사일 강화 시스템을 통해 자신만의 전투 스타일 구축.
*   **화려한 액션:** 위기 상황을 역전시키는 강력한 필살기와 박진감 넘치는 보스전.

**3. 게임 플레이**

*   **조작:**
    *   **이동:** 키보드 방향키(PC) 또는 화면 터치(모바일)로 고양이 비행기 조종.
    *   **공격:** 기본 미사일은 자동 발사.
    *   **필살기:** 전용 버튼을 눌러 발동 (게이지 충전 방식).
*   **스테이지 구성:**
    *   각 스테이지는 여러 웨이브의 일반 쥐 적들과 중간 보스, 그리고 최종 보스로 구성됩니다.
    *   스테이지가 진행될수록 새로운 적 패턴과 환경 요소(예: 소행성, 우주 잔해)가 추가됩니다.
*   **적 캐릭터:**
    *   **일반 쥐:** 다양한 이동 패턴과 공격 방식(직선, 곡선, 산개 등)을 가진 기본 적.
    *   **엘리트 쥐:** 일반 쥐보다 강력한 공격력과 체력을 가지며, 특수 공격 패턴을 사용.
    *   **보스 쥐:** 각 스테이지의 최종 목표. 거대한 크기와 독특하고 강력한 공격 패턴을 가지며, 특정 약점을 공략해야 합니다.
*   **주인공 기체 (고양이 비행기):**
    *   **기본 공격:** 전방으로 미사일 발사.
    *   **목숨:** 기본 3개로 시작. 체력이 0이 되면 목숨 1개가 소모되고 스테이지 처음부터 다시 시작.
    *   **체력:** 적의 공격에 피격 시 체력이 감소.
*   **아이템 및 강화:**
    *   **미사일 강화 아이템:** 적 처치 시 드롭되며, 획득 시 미사일의 공격력, 연사력, 발사 개수, 범위 등을 단계적으로 강화합니다. (예: 1단계 미사일 -> 2단계 미사일 -> 레이저 -> 확산탄 등)
    *   **보조 아이템:** 쉴드, 속도 증가, 체력 회복 등 일시적인 효과를 제공하는 아이템.
    *   **보너스 아이템 (치즈):** 스테이지에 등장하며 획득 시 목숨이 1개 늘어납니다.
*   **필살기:**
    *   전용 게이지가 가득 찼을 때 사용할 수 있으며, 사용 시 화면 전체 공격 또는 강력한 단일 대상 공격을 가합니다.
    *   사용 후 일정 시간 동안 쿨타임이 적용됩니다.
*   **보스전:**
    *   각 스테이지의 마지막에 등장하며, 보스마다 고유한 공격 패턴과 공략법이 존재합니다.
    *   보스를 처치하면 다음 스테이지로 진행됩니다.

**4. 승리 조건**

*   모든 스테이지의 최종 보스를 성공적으로 처치.

**5. 패배 조건**

*   가진 목숨을 모두 잃을 경우.

**6. 향후 개발 방향 (확장 가능성)**

*   **신규 기체 추가:** 다양한 종류의 고양이 비행기(각기 다른 기본 공격 및 필살기) 추가.
*   **멀티플레이 모드:** 친구와 함께 쥐 군단을 물리치는 협동 플레이 또는 경쟁 모드.
*   **랭킹 시스템:** 전 세계 플레이어들과 점수 경쟁.
*   **스토리 모드 강화:** 고양이와 쥐의 우주 전쟁에 대한 심화된 스토리라인 추가.
