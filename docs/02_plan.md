### **"냥냥 슈터" 개발 작업 계획**

**Phase 1: 핵심 게임 플레이 구현**
1.  **주인공 기체(고양이 비행기) 구현:**
    *   [x] 화면에 고양이 비행기 표시
    *   [x] 키보드 방향키로 비행기 이동 기능 구현
    *   [x] 기본 미사일 자동 발사 기능 구현
2.  **생명 및 체력 시스템 구현:**
    *   [x] 체력(Health) 시스템 구현 (적과 충돌 시 감소)
    *   [x] 목숨(Lives) 시스템 구현 (기본 3개, 체력 0 되면 1개 차감)
    *   [x] 목숨 소진 시 스테이지 처음부터 다시 시작하는 로직 구현
3.  **기본 적(일반 쥐) 구현:**
    *   [x] 화면에 일반 쥐 적 캐릭터 표시
    *   [x] 위에서 아래로 단순 이동하는 패턴 구현
4.  **충돌 감지 구현:**
    *   [x] 아군 미사일과 적의 충돌 감지 (적 파괴)
    *   [x] 적과 아군의 충돌 감지 (체력 감소)

3.  **게임 클리어/오버 조건 구현:**
