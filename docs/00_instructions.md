## instructions

1. 게임 기획서와 개발 작업 계획을 꼼꼼히 읽어주세요.
2. "냥냥 슈터" 게임 개발을 위해 필요한 기능들을 단계별로 구현해 나가야 합니다.
3. `02_plan.md` 파일에 각 단계별로 구현해야 할 기능들이 명확히 정의되어 있으니, 이를 참고하여 개발 작업을 진행해주세요.
4. 각 단계가 완료될 때마다 작업상황을 계속 업데이트 해주세요.
5. 이 프로젝트는 git 으로 관리되고 있으므로, 각 단계가 완료될 때마다 커밋을 해주세요.
6. 커밋은 Conventional Commits 규칙을 따르며, 예시로는 다음과 같습니다:
   - feat: 주인공 기체 이동 기능 구현
   - fix: 충돌 감지 버그 수정
   - docs: 게임 기획서 업데이트
